The app plays the sound of rain or the beach while you meditate.  You can choose to meditate for two, five or 10minutes.  

## Instructions

Clone the App
```
git clone git@github.com:LisaMcCormack/meditation-app.git
```

In the terminal:
```
open index.html
```
